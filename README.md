# Sass Template

This is a versatile template for organizing your Sass projects, providing a clear and efficient structure for styling.

-----

## How to use it

1.  Just download the `sass` folder.
2.  Include it in your project.
3.  Start using this Sass Template for your styling needs.

-----

## Folder structure

```
sass/
├── abstracts/
│   └── _index.scss
│   ├── _mixins.scss
│   ├── _variables.scss
├── base/
│   ├── _fonts.scss
│   ├── _global.scss
│   ├── _reset.scss
│   └── _index.scss
│   ├── _reset.scss
├── layouts/
│   └── _index.scss
├── components/
│   └── _index.scss
├── utils/
│   └── _index.scss
└── main.scss
```
